<h1>★ BookStoreApp</h1>
<img width="100%" src="https://github.com/JoshuaThadi/Mobile-Apps/blob/main/Mobile-Apps/bookstore/bookstore-1.png">

This full-stack application enables functionalities like browsing book lists, viewing details, managing favorites, and connecting to external resources — all in a responsive and user-friendly interface. BookstoreApp demonstrates the power of combining a cross-platform frontend with a scalable backend architecture, making it ideal for book lovers and tech enthusiasts alike.
